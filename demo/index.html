De quoi s'agit il?

Ce petit plugin vous permet d'aggrandir vos images au dessus de votre contenu à partir d'une seule image. Exemple: 


.
..
.

Comme vous pouvez le remarquer, l'image contient un attribut data-gallery qui permet de différencier les différentes galeries de votre page. Voici donc une autre galerie dans la même page mais avec un "data-gallery" différent:


.
..
.

Tout ce dont vous avez besoin c'est de jQuery et du script avec son initialisation à la fin de votre document.

$(document).jquerygallery({
// Si vous voulez les miniatures en dessous.
'coverImgOverlay' : true,
// La classe contenant les images pour la stylisation.
'imgActive' : "imgActive",
// La classe contenant les miniatures.
'thumbnail' : "coverImgOverlay",
// La hauteur que vous attribuer à ces minatures
'thumbnailHeight' : 120,
// Le nom de la classe qui sert à assombrir toute la page.
'overlay' : "overlay",
// Les icones d'actions. Ceux-ci viennent de fontAwesome.
'imgNext' : "",
'imgPrev' : "",
'imgClose' : "",
// La vitesse des animations.
'speed' : 300
});

Architecture du document

Dans le fichier téléchargé, vous pourrez retrouver plusieurs dossiers avec un fichier.txt explicatif. Cette démo est réalisée avec SASS, le fichier style.css est donc compressé. Tous les styles concernant le plugin en lui même se trouve dans le fichier 5-js.scss. Il est également compilé à la racine du document jquery-gallery.css
